# User Guide

## Client Dashboard
1. **Login** → Enter email/password.
2. **Create Page** → Click "Create" → Fill details → Save.
3. **Edit Page** → Click a page to make changes to icon → theme, or images or body content and Save.
4. **Publish Page** Click publish to publish the page.
4. **View Analytics** → Dashboard → Select a page → See daily, weekly, monthly charts.

---

## Admin Guide
- **View Global Analytics** → `/dashboard`

---

## Troubleshooting
- **Charts not loading** → Check network request to `/analytics`
- **File upload fails** → Ensure < 2MB and valid image format
- **No view tracking** → Verify tracking script is included in public page template
